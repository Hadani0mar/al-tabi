## [0.1.20] - 2026-06-02

### تحسينات جوهرية في الأداء وتقليل التوكنز
- تقليص استهلاك التوكنز بـ 70%: الوكيل في الوضع السريع يُرسل [system + السؤال] فقط بدل كامل تاريخ المحادثة
- إضافة Slim Summarize History: طلب التلخيص يُرسل 3 رسائل فقط بدل 8-10 رسائل سابقاً
- ضغط نتائج الاستعلامات في التاريخ: الحفاظ على 8 صفوف فقط مع البيانات الوصفية
- حذف أداتَي get_current_datetime وlist_available_patterns من الأدوات (التاريخ محقون في الـ prompt)
- النموذج لا يحتاج جولة API إضافية لمعرفة التاريخ → رحلة واحدة بدل رحلتين
- حد تاريخ المحادثة في الواجهة: آخر 6 رسائل فقط
- إصلاح حقن مسار ملفات PDF/Excel تلقائياً حتى عند ردود النموذج القصيرة

### جديد
- إضافة مكتبة OpenAI SDK للواجهة الأمامية (عبر OpenRouter) مع دعم Streaming
- Streaming كامل للردود النصية: النص يظهر حرفاً بحرف مع cursor وامض بدل الانتظار
- ضغط system prompt من 2665 حرفاً إلى أقل من 2000 حرف مع اختبارات تلقائية
- توجيه صريح للبحث في التواريخ القديمة عبر days_recent=N أو @TargetDate
